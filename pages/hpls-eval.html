HPLS-Eval research axis

HPLS-Eval extends the project with precision, runtime, hardware, and domain-shift observations while keeping the LiteRaceSegNet core claim intact.

Work package 1

Define case-set manifest, preprocessing, threshold policy, and output artifact contract.

Work package 2

Run FP32 reference path and preserve masks, logs, runtime backend, and fallback status.

Work package 3

Compare FP16 and INT8 target outputs against FP32 reference masks when measured.

Work package 4

Extend the protocol to CPU, GPU, possible NPU, and possible TPU without claiming untested hardware behavior.
