Historical reference rule

Imported historical results must keep their original context and must not be relabeled as v15.2 measurements. Use them only as reference evidence with clear provenance.

Result status labels

  • planned: protocol exists, no result yet.
  • measured_case: case-level masks and logs are present.
  • measured_set: multiple cases with aggregated summary.
  • future_protocol: planned for hardware/runtime not yet tested.

Public package note: this portal contains documentation, selected source references, protocols, and result templates for review.
