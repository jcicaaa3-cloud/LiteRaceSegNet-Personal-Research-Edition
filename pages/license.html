License and release boundary

Public release scope

This package is meant for research-portfolio review and protocol documentation. It is not a deployed inference product.

Public package includes

Source references, GitHub Pages files, selected diagrams, public documentation, extension protocols, result templates, and metric helper scripts.

Public package excludes

Private datasets, checkpoints, pretrained weights, credentials, local environment artifacts, cloud keys, and browser inference.
