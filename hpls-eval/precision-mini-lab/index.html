What this page is for

Use this page as the practical entry point for small controlled precision checks. It gives reviewers a clear path from planned comparison rows to recorded tables and simple mask-level measurements.

What is included

  • Experiment matrix for planned comparison rows.
  • Result template for case-level and aggregate records.
  • Metric helper for reference-versus-target mask comparison.
  • Table and note templates for later result collection.

Suggested reading order

  1. Open the experiment matrix.
  2. Review the result template.
  3. Use the metric helper if mask comparison is needed.
  4. Return to HPLS-Eval for the broader evaluation direction.
