# Public Overview

LiteRaceSegNet Personal Research Edition is a static research-portfolio package for lightweight road-damage segmentation and follow-up output-stability analysis.

The public portal is organized around:

- LiteRaceSegNet core architecture;
- HPLS-Eval as a separated follow-up program;
- Precision Mini-Lab as a pilot FP32 / FP16 / INT8 protocol;
- output-stability metrics;
- domain and hardware shift planning;
- claim-boundary discipline.

The package does not include raw private datasets, checkpoints, cloud credentials, upload flow, API inference, or browser-side model execution.
